### Available scripts

- Clone this repository
```bash
$ git clone git@github.com:leonardodiasb/final-capstone-back-end.git
$ cd final-capstone-back-end/
```

- Seed the database
```bash
$ rails db:seed
```

- Start Rails server
```bash
$ rails s
```

- Start Rails console
```bash
$ rails c
```
